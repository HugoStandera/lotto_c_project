# Lotto C Project

A simple console-based lottery programme written in C.  
Generates random lottery numbers and checks them against user input.

## Features

- Generates a set of random lottery numbers.
- Allows user to input their own numbers or have them generated.
- Compares user numbers against the winning numbers.
- Displays how many numbers the user matched.

## How to Compile

Make sure you have a C compiler installed (e.g. Dev-C++).  
Open a terminal in the project folder and run.
